# cn-depu-guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要存放独立的 HTML 页面文件，这些页面可用于展示、参考或作为特定场景下的辅助工具。仓库不针对任何单一域名或网站，所有页面均为独立维护的静态文件。

## 目录说明

```
cn-depu-guide/
├── index.html          # 仓库主页 / 导航页
├── pages/              # 存放各类独立页面的文件夹
│   ├── example.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的默认入口页面，可提供页面列表或引导信息。
- `pages/`：所有独立 HTML 页面均归档于此，每个页面可独立运行。
- `assets/`：存放页面公用的样式、脚本及媒体资源。

## 页面归档说明

- 每个 HTML 页面应尽量自包含，若需引用外部资源，请使用相对路径指向 `assets/` 目录。
- 页面命名建议使用有意义的英文或拼音，避免使用特殊字符。
- 如需新增页面，请在 `pages/` 下创建新文件，并确保其可正常访问。
